## Installation

### Prerequistes

- Node.js 12.x LTS Erbium
- NPM

### 1. Clone

```shell
$ git clone 
```

### 2. Install Packages

```bash
$ npm install
```

### 3-1. Start development server

```shell
$ npm run dev
```

After starting the server,
go to [`http://localhost:3000`](http://localhost:3000)

### 3-2. Start production server

```shell
$ npm build 
```
